#pw 模块
---

PW全名即为 "PixelsWorld"。一个可以运行AE某些功能的工具箱。

##函数成员

- pw.getColor(x,y)
- pw.setColor(x,y,R,G,B,A)
- pw.ch(name,id)
- pw.in2out()
- pw.glsl(glslCode)
- pw.full_glsl(full_glslCode)
- pw.shadertoy(shadertoyCode)
- pw.js(jsCode)
- pw.cmd(cmdCode)
- pw.lua(luaCode)
- pw.report(message)

##数组成员

- pw.debug
