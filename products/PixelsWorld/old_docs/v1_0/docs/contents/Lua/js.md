#pw.js(jsCode)
---

**Description :**

- Run JavaScript code. 

**Parameters :**

- pw.js(jsCode). 
- jsCode : string

**Return :**
- Its first return is the resault of js code. 
- Its second return is the error message.  

**Sample code :**
```lua:js.lua
pw.js("alert('Hello PixelsWorld!');")
```
