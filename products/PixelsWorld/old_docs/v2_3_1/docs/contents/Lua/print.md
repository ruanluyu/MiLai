# pw.print()
---

**Description**

- Print string into screen (only ASCII is supported). 


**Parameters**

- pw.print(str)
- pw.print(str,alpha,red,green,blue)
- str: Input message
- alpha: Font alpha
- red: Font red
- green: Font green
- blue: Font blue

**Return**

- None

**Example**

```lua:print.lua
pw.print("Hello!");
pw.print("Hello!",1,0.5,0.5,1);
```
