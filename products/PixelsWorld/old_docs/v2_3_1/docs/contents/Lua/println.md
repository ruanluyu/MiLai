# pw.println()
---

**Description**

- Print string into screen (only ASCII is supported). 
- Add "\n" automatically at the end. 


**Parameters**

- pw.println(str)
- pw.println(str,alpha,red,green,blue)
- str: Input message
- alpha: Font alpha
- red: Font red
- green: Font green
- blue: Font blue

**Return**

- None

**Example**

```lua:println.lua
pw.println("Hello!");
pw.println("Hello!",1,0.5,0.5,1);
```
