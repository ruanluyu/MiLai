# pw.alert()
---

**Description**

- Add a yellow alert message into your screen. Equals to `pw.println(str,1,1,1,0)`


**Parameters**

- pw.alert(str)
- str: Alert string

**Return**

- None

**Sample code**

```lua:alert.lua
pw.alert("Hello!");
```
