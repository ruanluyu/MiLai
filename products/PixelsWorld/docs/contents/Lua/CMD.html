CMD Code

CMD Code

CMD commands can be executed by function pw.cmd("Your cmd commands") on Lua (CPU & GPU) language mode.

executeCMD.lua

//Works on Windows platform
pw.cmd("echo Hello PixelsWorld! & pause");
