#pw.debug.add(message)
---

**Description :**

- Insert a message into the debug window. 
> Note : this is exactly same with [```pw.report()```](report.md)


**Parameters :**

- pw.debug.add(message)
- message : string

**Return :**
- No return.

**Sample code :**
```lua:pw_debug_add.lua
pw.debug.add("Message from time "..time.."\nHello PixelsWorld!\n")
```
