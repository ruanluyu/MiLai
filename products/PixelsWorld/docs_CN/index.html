PixelsWorld是什么

什么是PixelsWorld?

PixelsWorld 是一款通过脚本来渲染图像的插件。 您几乎可以用它来渲染任何效果：

  • 可以渲染 GLSL
  • 可以执行 Lua
  • 可以执行 JavaScript
  • 可以执行 CMD

PixelsWorld 文档版本 : V1.0
