uv2xy,xy2uv

uv2xy,xy2uv

这两个函数用于UV坐标与 XY 坐标之间转换 只是做一些乘除法。

uv2xy(vec2)

说明：

  • 通过给定的uv值计算xy值。

参数：

  • vec2 : uv位置值。

返回：

  • vec2 : xy位置值。

注意： 如果图层尺寸为1280 x 720，则 uv2xy(vec2(0.1,0.5)); 返回 vec2(128.0,360.0);

xy2uv(vec2)

说明：

  • 通过给定的xy值计算uv值。

说明：

  • vec2 : xy位置值。

返回：

  • vec2 :uv位置值。

注意： 如果图层尺寸为1920 x 1080，则 xy2uv(vec2(192,108)); 返回 vec2(0.1,0.1);
