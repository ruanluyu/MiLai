◆ pixels_per_meter

double wavefd::WaveFDData::pixels_per_meter

How manny pixels are there in an 1 meter 1D fluid field.

20 means there are 400 pixels in an 1 square metre 2D fluid field.

Set it before calling wavefd::create.

You are not supposed to change it after calling wavefd::create.

You can change it after calling wavefd::dispose.

◆ sample_rate

unsigned int wavefd::WaveFDData::sample_rate

This will be calculated by WaveFD, and it is based on the width and height you have sent to wavefd::create.

This result is also based on CFL condition.

It means the minimum sample rate we can set to make sure the accuracy of the simulation.

You are not supposed to change it.

Will be assigned after calling wavefd::create.

Will be reset after calling wavefd::dispose.

◆ current_sample_id

unsigned int wavefd::WaveFDData::current_sample_id

You are not supposed to change it.

Will be assigned to 0 after calling wavefd::create.

Will be reset after calling wavefd::dispose.
