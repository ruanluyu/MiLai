◆ VisualizationFlagEnum

enum wavefd::VisualizationFlagEnum

Used to set WaveFDData::visualization_flag.

See also
flagtools
Enumerator
VISUALIZATION_ON 

The window updates or not. True: update, flase: otherwise.

VISUALIZATION_SHOW_BARRIERS 

Show the barriers or not. True: show, false: otherwise.

VISUALIZATION_SHOW_SOUND_WAVES 

Show the sound waves or not. True: show, false: otherwise.

VISUALIZATION_SHOW_EXCITATION 

Show the excitation mechanisms or not. True: show, false: otherwise.

VISUALIZATION_FULL 

True: render each simulated result. False: base on WaveFDData::visualize_fps.

◆ force_visualize()

API int wavefd::force_visualize ( WaveFDData &  wavefd_data)

Visualize sound field one frame forcibly.

Almostly, you don't have to call this function by yourself.

The wavefd::simulate function will call this function automatically.

This function ignores the flag wavefd::VISUALIZATION_ON.
